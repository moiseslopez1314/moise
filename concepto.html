HTML

HTML es un lenguaje de programacion que permite crear paginas web en un sistema de etiquetas que cada una tiene una funcion, estas etiquetas cuentan con una apertura y un cierre , asi tambien una interfaz donde va el contenido. HTML significa lenguaje de marcas de hipertexto , que quiere decir todo va unido por mayormente links. HTML tiene etiquetas para texto, tambien etiquetas que no deben faltar, de tablas, de cracion de formularios y de multimedia entre otros.
